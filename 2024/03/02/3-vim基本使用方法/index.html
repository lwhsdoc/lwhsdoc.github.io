3.vim基本使用方法

vim简单入门

2024年3月2日

3:09

虽然说大部分发行版默认编辑器不是vim而是nano，但是他的通用性很好，在Linux和FreeBSD等等系统上都可以用。

这里介绍他最基本的使用方法↓

首先，你要确定你的系统有没有安装vim

vim -v

如果有反应，并且他打开了一个显示版本号的页面，你可以输“：q”退出，如果报错提示“command not found”就请安装它

Debian/Ubuntu↓

sudo apt install vim -y

rhel/alma/rocky因为自带vim所以说不需要单独安装

使用方法↓

vim默认情况下vim是命令模式，按下i进入编辑模式以后你就可以像普通的文本编辑器编辑文本。

保存、退出或者是保存后退出或者是强制退出需要进入命令模式，从编辑模式退出到命令模式按下esc退出。

命令模式下输入退出vim::q（如果你对文本做了更改他会报错）

保存更改输入:w

保存更改并退出输入:wq

强制退出不保留更改输入:q!

请注意，仅仅适用于最基本的使用，仅仅适用于日常编辑文本的工作。提醒一句，如果你要查看重要文本，而不想对他做出任何的整改，防患于未然，请用view。
